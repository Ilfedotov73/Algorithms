# Education C++ project for computer science.
This is a project based on template for C++ projects which utilise a Core/App project architecture. 
[Premake](https://github.com/premake/premake-core) is used to generate project files.

The `Scripts/` directory contains build scripts for Windows and Linux, and the `Vendor/` directory contains Premake binaries (currently version `5.0-beta2`).
